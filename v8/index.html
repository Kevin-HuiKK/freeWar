    2. 基本行动（基础 2 点行动）

    • 移动：将部队从一个己方城市移动到相邻连接城市。
    • 占领：攻击相邻敌方城市，战斗胜利后占领。
    • 建设：在己方城市建立道路或港口海路。
    • 贸易：通过港口或连接路线获得资源与影响力。
    • 招募：消耗资源招募部队，不消耗行动点。

    3. 战斗机制

    1. 攻击方从相邻己方城市发动攻击。
    2. 道路 / 海路决定可攻击目标。
    3. 要塞和主城拥有额外防御。
    4. 切断路线可让敌方城市孤立。

    4. 资源系统

    类型获取方式用途
    金币城市与贸易建路、招兵、升级
    劳力城市生产建设、修复、攻城
    影响力主城网络外交、视野与后续扩展

    5. 特殊规则

    • 影响范围由城市等级和类型决定。
    • 连接主城的城市获得完整收益。
    • 孤立城市收入和成长降低。
    • 海路必须连接两个港口。

    6. 回合流程

    1. 资源收入
    2. 城市成长
    3. 敌方行动
    4. 胜利检查
    5. 恢复基础 2 点行动，天赋可提高上限

    7. 势力说明

    战报 / 胜利奖励

    占领 3 个首都胜利后领取天赋点，用来升级主城、道路和港口工程。
